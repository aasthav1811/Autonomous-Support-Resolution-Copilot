# Technical FAQ

## The app keeps crashing
1. Update to the latest version from the app store
2. Clear app cache: Settings > Apps > [App Name] > Clear Cache
3. Restart your device completely
4. If the issue persists, send crash logs to support@company.com with your device model and OS version

## I cannot log in
1. Check that Caps Lock is off
2. Use "Forgot Password" to reset your password
3. Clear your browser cookies and cache
4. Try incognito or private browsing mode
5. If 2FA is enabled, ensure your authenticator app time is synced correctly

## Sync is not working
Sync requires an active internet connection. Pull down on the home screen to force a sync. Check Settings > Account > Last Sync time to see when data last synced successfully.

## API returning 500 errors
Check status.company.com for any ongoing incidents. Verify your API key is active in the dashboard. The rate limit is 1000 requests per minute for standard plans. If the issue persists, contact our API support team.

## Feature is missing from my account
Some features are rolled out gradually. Check Settings > About for your version number and review our changelog at company.com/changelog to see when features are available.

## How do I clear the app cache?
On iOS: Delete and reinstall the app. On Android: Settings > Apps > [App Name] > Storage > Clear Cache. On desktop: Help > Clear Cache or reinstall.
